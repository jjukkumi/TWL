# Chee-up!

취업을 하기위한 2G의 공부 레포

### Rule

#### how to

1. 우리는 시작 시간이 정해져 있지 않고 유연 공부제로 한다.
2. 공부시간을 매일 7시간을 원칙으로 한다. (평일)
3. 공부를 시작하면 처음에 공부할 것을 정리해서 pr를 날린다.
4. 모든 파일은 .md 파일을 원칙으로 한다.
5. 각자 이름으로 된 브랜치를 생성한뒤 pr을 날린다.
6. 그 다음날 확인한 뒤 master에 merge 한다.
